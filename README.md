# Quiz React App

This project is a quiz application built with React and TypeScript.
It focuses on the core quiz flow such as selecting answers, checking correctness, and displaying results.

The app is designed to practice component-based architecture, state management, and clean UI composition.
It uses a scalable structure so that additional features and quiz logic can be extended incrementally.

---

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
